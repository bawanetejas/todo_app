# Todo Application

A lightweight Todo Application built using HTML, CSS, and Vanilla JavaScript as a refresher project to revisit core frontend fundamentals and DOM manipulation concepts.

---

# Purpose

This project was created to refresh and strengthen foundational frontend concepts such as:

- DOM manipulation
- Event handling
- Dynamic rendering
- State management using arrays
- Basic UI structuring with HTML and CSS

Instead of using frameworks, the project focuses on understanding how things work internally using plain JavaScript.

---

# Features

- Add todos dynamically
- Render tasks in real-time
- Clean and responsive UI
- Simple state management
- Interactive user input handling

---

# Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
